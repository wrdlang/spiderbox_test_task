## TechStack
This is a SPA, laravel on the backend and ReactJS for the frontend, with InertiaJS to connect them (as a state management).

- Php/Laravel
- ReactJS
- InertiaJS

## Run the project
Run using docler compose:
- docker-compose run laravel.test
or using laravel sail:
- ./vendor/bin/sail up
